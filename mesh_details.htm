Mesh Details

Use Mesh Details to control element/node label numbers, and to change group names for each map in the assembly FEM. The software automatically creates a group for each instance in the map. The group contains copies of all the nodes, elements, and coordinate systems of the component FEM.

Once you append the map to the assembly FEM, you can't change a group name or label. If you're not satisfied with label numbering after the append, delete the map from the assembly FEM and re-enter it with new label numbers.

To modify group names or label numbers:

  1. Decide whether you want to use a start label and increment or an just an offset for the labels.
    • Pick Use label increment to specify an increment for the labels.
    • Pick Use label offset to specify an offset for all labels.

  2. Set the form to display node, element, or physical property table data.

    Note: You can only display physical property table data if you picked  Create unique physical property tables on the Create FEM from Assembly form.

  3. Pick the assembly item/FEM map to modify.

  4. In the key-in fields below the table, enter a new group name, start label, and increment or offset.

  5. Pick the green Go button (or Enter) to apply the changes.

    The software checks for label conflicts with data already in the assembly FEM. If it finds conflicts, it reports them and resets the fields to their original values.

    If there are no conflicts, you'll see a value in the  End Label column.

  6. Continue to modify other maps.

  7. When changes are complete, pick OK to return to Create FEM from Assembly.

  8. Pick Update to append the maps to the assembly FEM.

  9. Pick Mesh Details again to review the label numbers. They will be updated if you previously used defaults.

Options that need some explanation:

  • Use Label Increment/Use Label Offset lets you specify either a start label and increment or an offset to use to control node, element, or physical property table labels.

  • If you pick Use Label Offset, Default to nearest lets you define the default offset for all maps.

  • Toggle on Element labels same as nodes to make the start labels and increments or offsets the same for elements and nodes.

  • If you pick Use Label Increment, the start label is 0. If you don't set a value, the software will then take the default, which is the next highest label available in the model. If you set a start label and then want to revert back to the default, just set the start label back to 0.

    After you have appended the component FEMs to the assembly FEM, the form shows the actual labels used.

  • If you pick Use Label Increment, the label increment (Incr.) is 0. If you don't set a value, the software will use the Default node (or element) label increment value. As with Start Label, you can always reset a modified increment to the default by entering 0.

    Once you have appended the component FEMs, Incr. will show the actual increment used.

  • Group-name prefix lets you change the prefix style for group names. The software automatically assigns group names, and applies a letter or instance hierarchy number as a prefix when a map uses multiple instances.

    If you use the Letter option, the letter is assigned based on the instance count. That is, the third instance of the assembly item always gets the letter C, even if the first and second instances are not used in any maps.
